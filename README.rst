dt-devsetup-plugin plugin for `Tutor <https://docs.tutor.overhang.io>`__
===================================================================================

Installation
------------

::

    pip install git+https://github.com/Dicey-Tech/dt-devsetup-plugin

Usage
-----

::

    tutor plugins enable dt-devsetup-plugin
